# Innovation Points

## 1. 时空一致的轨迹建模框架

我们将轨迹数据统一建模为空间几何、时间节律和历史模式的组合，不仅关注点坐标本身，也显式利用时间戳、速度、方向和轨迹形状信息。两个任务共享一套底层特征库（Haversine距离、速度、方位角变化、轨迹曲折度）。

## 2. 局部缺口模板残差修复（Task A）

Task A 最终方法不是整条轨迹模板匹配，而是对每个相邻已知点之间的缺口单独建模。我们从训练集中检索相同 span、局部位置和端点位移相似的短片段，学习这些历史片段相对直线插值的弯曲残差，再叠加到当前缺口上。

**可讲要点**：此前整条轨迹 KNN 模板明显失败，原因是全局 OD 相似无法保证中间局部缺口对齐。新方法把相似性粒度降到“缺口级”，因此能利用历史路网形状而不需要显式 OSM 路网。

验证集效果：
- 1/8：MAE 64.73m，RMSE 92.02m
- 1/16：MAE 120.73m，RMSE 168.34m

## 3. 保形插值与历史残差的两阶段结构（Task A）

PCHIP 作为无训练数据 fallback，避免普通三次样条在稀疏点间过冲；局部模板方法只在训练集有相似局部片段时进行残差修正。这个结构兼顾稳定性和历史先验。

**可讲要点**：PCHIP 从 Catmull-Rom 的 89.16m / 166.22m MAE 进一步降到 87.59m / 163.53m，局部模板再降到 64.73m / 120.73m。

## 4. 多层特征驱动的 TTE 回归（Task B）

从距离、轨迹形状、出发时间和空间区域四个层面构建特征，使模型能够同时捕捉路径长度、绕行程度、交通时段和城市区域差异。

**可讲要点**：tortuosity（曲折度）和bearing_change等形状特征是该任务中不常见但有效的建模角度。

## 5. 规则估计与残差学习融合（Task B）

先用分时段速度模型构建物理可解释的基础估计，再训练回归模型学习其残差，兼顾物理解释性和非线性拟合能力。

**可讲要点**：残差目标比直接预测 travel_time 更集中，模型学习重点在于解释拥堵、绕行等带来的偏差。

## 6. 可复现实验管理

每个方法都有配置文件、指标CSV、实验日志，便于对比分析和报告撰写。
